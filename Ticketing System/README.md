# Ticketing System Labs

This project demonstrates the use of best practices when addressing tickets via the online ticketing system, Spiceworks.

## What's Covered

1. Spiceworks Setup
2. Ticket Creation
3. Ticket Closure Checklist
4. Ticket Escalation
